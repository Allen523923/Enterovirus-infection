# Enterovirus-infection
- Taiwan Enterovirus Infection Statistics
- 腸病毒疫情統計與自動化分析工具。
- 本專案旨在分析台灣地區腸病毒的傳播趨勢。

# 腸病毒資料統計
- 自動抓取政府公開資料（Open Data）。
- 1.每年腸病毒就診人次畫成長條圖
- 2.每週腸病毒就診人次畫成折線圖
- 3.腸病毒就診類別畫成圓餅圖
- 4.年齡別腸病毒就診人次畫成長條圖
- 5.各縣市腸病毒就診人次畫成長條圖 , 2格
- 6.六都與非六都腸病毒就診人次佔比畫成圓餅圖
- 7.各六都腸病毒就診人次比率畫成圓餅圖

# 資料來源 (Data Source)
台灣政府資料開放平台 - 健保門診及住院就診人次統計-腸病毒
